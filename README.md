## Como Executar os Códigos

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/logica-algoritmos-java.git
```

Acesse a pasta do projeto:

```bash
cd logica-algoritmos-java
```

Compile o código:

```bash
javac NomeDoArquivo.java
```

Execute:

```bash
java NomeDoArquivo
```
